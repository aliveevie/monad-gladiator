# $GLAD Token Plan — MonadGladiator Arena Token

## Overview
**$GLAD** (Gladiator) is the governance and utility token for the MonadGladiator gaming arena ecosystem on Monad.

## Token Details
- **Name**: Gladiator
- **Symbol**: GLAD
- **Chain**: Monad (Chain ID 143 mainnet / 10143 testnet)
- **Launch Platform**: nad.fun (Monad-native fair launch)
- **Total Supply**: 1,000,000,000 GLAD

## Utility

### 1. Fee Discounts
- Hold 10,000+ GLAD → 25% discount on arena fees (2.5% → 1.875%)
- Hold 100,000+ GLAD → 50% discount (2.5% → 1.25%)
- Hold 1,000,000+ GLAD → 0% fees (free arena access)

### 2. Governance
- Vote on new game types to add to the arena
- Vote on fee rate changes
- Vote on tournament structures and prize pools
- 1 GLAD = 1 vote

### 3. Revenue Share
- 50% of arena fees distributed to GLAD stakers weekly
- Pro-rata based on staked amount
- Auto-compound option available

### 4. Arena Features
- GLAD holders can create private tournaments
- Exclusive game modes for holders
- Priority matchmaking
- Custom gladiator names/titles (NFT integration planned)
